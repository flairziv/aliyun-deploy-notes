# 2. 本地打包

## 准备工作

### 安装 Maven

项目如果没有自带 Maven Wrapper（`mvnw` 文件），需要手动安装：

1. 下载：https://maven.apache.org/download.cgi
2. 解压后将 `bin` 目录添加到系统 PATH 环境变量
3. 验证安装：

```cmd
mvn -v
```

## 后端打包

```cmd
cd ai-hardware-server
mvn clean package -DskipTests
```

- `clean`：清除上次编译产物
- `package`：编译并打成 jar 包
- `-DskipTests`：跳过单元测试（加快打包速度）

打包成功后，jar 文件在：

```
ai-hardware-server\target\ai-hardware-server-0.0.1-SNAPSHOT.jar
```

> **踩坑**：如果打包报错，先检查 `pom.xml` 里的 Java 版本和本地 JDK 版本是否一致（本项目要求 Java 21）。

## 前端打包

```cmd
cd ai-hardware-ui
npm install
npm run build
```

- `npm install`：安装依赖（第一次或 `package.json` 有改动时需要）
- `npm run build`：执行 Vite 构建，生成生产环境代码

打包成功后，静态文件在：

```
ai-hardware-ui\dist\
```

`dist` 目录里通常包含：
- `index.html` — 入口页面
- `assets/` — JS、CSS、图片等资源（带 hash 文件名用于缓存）

## 文件上传工具

推荐使用 **WinSCP**（图形界面 SFTP 工具）上传文件到服务器：

- 下载：https://winscp.net
- 连接方式：SFTP 协议
- 填入：服务器 IP、端口 22、用户名 root、密码

**上传清单**：

| 本地文件 | 服务器路径 |
|---------|-----------|
| `target\ai-hardware-server-0.0.1-SNAPSHOT.jar` | `/opt/ai-hardware/app.jar` |
| `dist\` 目录下所有文件 | `/var/www/ai-hardware/` |
| `public\images\` 下所有图片 | `/var/www/ai-hardware/images/` |
| `schema.sql` | `/opt/ai-hardware/schema.sql` |
