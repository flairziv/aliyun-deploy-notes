# 3. 服务器环境搭建

## 连接服务器

使用 SSH 登录：

```bash
ssh root@你的服务器IP
```

工具选择：
- **Windows Terminal**（Win11 自带）
- **PuTTY**（经典 SSH 客户端）

## 更新系统

```bash
apt update && apt upgrade -y
```

- `apt update`：更新软件包列表
- `apt upgrade -y`：升级已安装的软件包，`-y` 自动确认

## 安装 Java 21

```bash
apt install openjdk-21-jdk -y
java -version
```

验证输出应包含 `openjdk version "21.x.x"`。

> **为什么是 Java 21**：项目 `pom.xml` 中 `<java.version>21</java.version>`，服务器和本地的 Java 版本必须匹配。

## 安装 MySQL 8.0

```bash
apt install mysql-server -y
systemctl start mysql
systemctl enable mysql
```

- `systemctl start`：立即启动服务
- `systemctl enable`：设置开机自启

验证：

```bash
systemctl status mysql
```

看到 `active (running)` 就表示正常。

## 安装 Nginx

```bash
apt install nginx -y
systemctl start nginx
systemctl enable nginx
```

验证：在浏览器访问 `http://你的服务器IP`，看到 "Welcome to nginx!" 页面说明安装成功。

## 创建项目目录

```bash
# 后端 jar 包和上传文件目录
mkdir -p /opt/ai-hardware/uploads

# 前端静态文件目录
mkdir -p /var/www/ai-hardware

# 生产配置文件目录
mkdir -p /etc/ai-hardware
```

## 关于 systemctl 的常用命令

| 命令 | 作用 |
|------|------|
| `systemctl start 服务名` | 启动服务 |
| `systemctl stop 服务名` | 停止服务 |
| `systemctl restart 服务名` | 重启服务 |
| `systemctl status 服务名` | 查看服务状态 |
| `systemctl enable 服务名` | 设置开机自启 |
| `systemctl disable 服务名` | 取消开机自启 |
| `journalctl -u 服务名 -f` | 实时查看服务日志 |
